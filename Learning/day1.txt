MongoDB Day 1 Learning

Topics Covered:
- Database creation using "use"
- Viewing databases using "show dbs"
- Creating collections
- Renaming collections
- Dropping collections and database
- insertOne() and insertMany()
- find() to retrieve data
- BSON data types (string, number, boolean, null, array, object, date)

Commands Learned:

1. Create Database
use school

2. Show Databases
show dbs

3. Create Collection
db.createCollection("students")

4. Rename Collection
db.students.renameCollection("student")

5. Drop Collection
db.library.drop()

6. Drop Database
db.dropDatabase()

7. Insert One Document
db.student.insertOne({name:"Asif Khan", rollno:29, class:"ECS"})

8. Insert Many Documents
db.student.insertMany([
  {name:"Maaz Khan", rollno:28, class:"ECS"},
  {name:"Mashuk Khan", rollno:31, class:"ECS"},
  {name:"Zaid Khan", rollno:32, class:"ECS"}
])

9. Find Documents
db.student.find()

10. Insert Complex Document
db.personal.insertOne({
  name:"Asif Khan",
  age:45,
  married:true,
  weight:72.50,
  kids:null,
  hobbies:["music","sports"],
  address:{street:"123 feet road", city:"Mumbai"},
  dob: ISODate("2000-10-15T08:00:00Z")
})

Done!
